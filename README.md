# 股票趋势分析系统 v2.0

A股趋势票分析与全市场扫描系统，专为做趋势的投资者设计。

## 核心特性

### 主力侦察五问
对每只股票进行完整的"侦探式分析"：

1. **是否有主力？** - 通过换手、振幅、大单占比综合评估
2. **主力性质？** - 机构主导 / 游资主导 / 混战 / 散户为主
3. **控盘程度？** - 筹码集中度 + 大单稳定度 + 控盘指标
4. **主力在干什么？** - 11个细分阶段，每个阶段有明确动作描述
5. **建议如何操作？** - 精确的买卖区间、止损止盈位

### 阶段识别（11个细分阶段）
- 吸筹期（早期/后期）
- 启动期
- 拉升期（初期/中期/加速期）
- 洗盘期
- 派发期（早期/中期）
- 杀跌期
- 底部反转苗头
- 过渡期

### 4级机会分级
- 🟢 A级·拉升加仓 / 右侧建仓（确定性高）
- 🟠 B级·吸筹试仓（中等确定性）
- 🔵 C级·左侧观察（仅观察不入场）

### 全市场扫描器
8组筛选条件 + 5个内置预设：
1. 主力情况（性质、控盘度）
2. 主力阶段（含进入第N天）
3. 板块联动（板块强度+共振要求）
4. 大盘环境
5. 技术面（多周期、距离均线、量价）
6. 资金维度（多窗口、超大单要求）
7. 质地过滤（市值、ST、负债等）
8. 机会等级与综合评分

### Web 界面（黑底现代风格）
- 📊 总览 Dashboard
- 📈 大盘多指数（13种浪型识别）
- 🔥 板块4窗口排名
- 🎯 个股扫描（核心）
- ⭐ 自选股管理
- 📁 历史归档
- ⚙️ 设置

## 技术栈

- **后端**：FastAPI + SQLAlchemy 2.0 + APScheduler
- **前端**：Bootstrap 5 (Dark) + Alpine.js + DataTables + ECharts
- **数据库**：MySQL 5.7
- **数据源**：Tushare Pro
