## 一、核心理念

### 设计哲学

> **股票涨跌的本质 = 资金推动 × 趋势惯性。**

不试图猜"主力心理"，只识别"主力行为留下的客观痕迹"。

### 五面共振才是高确定性

| 维度 | 单独胜率 | 叠加效应 |
|---|---|---|
| 个股阶段对 | ~55% | 基础 |
| + 板块共振 | +10% | 65% |
| + 大盘上行 | +7% | 72% |
| + 主力机构 | +6% | 78% |
| + 技术回踩 | +4% | 82%+ |

**多维度共振 = 信号过滤**，每加一个条件就过滤掉一批假信号。

### 四个绝对原则

1. **熊市不做趋势票**（择时比择股重要）
2. **不在均线远离时追高**（守纪律比贪婪重要）
3. **跌破止损位无条件离场**（认错比对错重要）
4. **吸筹期不入场**（时间比成本重要）

## 二、大盘判断逻辑

### 2.1 多指数对比

监控 **10 个主要指数**：

```
000001.SH  上证综指
399001.SZ  深证成指
000016.SH  上证50      ← 大盘蓝筹
000300.SH  沪深300
000905.SH  中证500
000852.SH  中证1000     ← 中小盘
932000.CSI 中证2000     ← 小微盘
399006.SZ  创业板指
399673.SZ  创业板50
000688.SH  科创50
```

每个指数都计算今日 / 5日 / 10日 / 20日 涨跌幅。

### 2.2 13 种浪型识别（自动）

形态识别**全部基于客观规则**，不数浪。

| 形态 | 识别规则 |
|---|---|
| **底部反转启动期** | 刚突破 MA20 / 还在 MA60 下方但 4 周涨幅 > 5% |
| **主升浪初期** | 周线多头排列 + MA60 上行 + 持续不到 8 周 |
| **主升浪中期** | 多头排列 + MA60 上行 + 持续 8-24 周 + 12 周涨幅>10% |
| **主升浪加速期** | 多头排列 + 4 周涨幅 > 10% + 创新高 |
| **主升浪末期** | 多头但近高点 + 4 周涨幅 < 5% + 12 周涨幅 > 15% |
| **顶部构筑期** | 在前高附近 + MA20 还在涨 + 4 周涨幅 < 2% |
| **顶部确立** | 跌破 MA20 + MA20 仍 > MA60 + 距前高 -5% ~ -15% |
| **趋势下跌期** | 空头排列 + MA60 下行 + 距前高 < -10% |
| **加速杀跌期** | 趋势下跌 + 4 周跌幅 > 8% |
| **杀跌末期** | 距前高 < -20% + 日均振幅 < 2.5% + 4 周跌幅 < 3% |
| **底部构筑期** | 距前高 < -15% + 12 周涨跌幅 < 5% + 振幅 < 3% |
| **中级整理** | 多头排列中 + 距前高 -3% ~ -10% + 26 周涨幅 > 10% |
| **大箱体震荡** | 以上都不符合（默认） |

### 2.3 市场风格识别

对比**大盘股 vs 小盘股**、**价值 vs 成长**的近 20 日表现：

| 风格 | 判定条件 | 含义 |
|---|---|---|
| 小盘成长牛 | 小盘 > 大盘+3% 且 创业板 > 上证50+2% | 资金偏好题材成长 |
| 小盘活跃 | 小盘 > 大盘+3% | 题材机会多 |
| 大盘价值牛 | 大盘 > 小盘+3% 且大盘上涨 | 蓝筹白马为主 |
| 成长占优 | 创业板 > 上证50+3% | 成长股相对强势 |
| 全面下跌 | 所有指数 < -2% | 系统性风险 |
| 横盘震荡 | 所有指数绝对值 < 2% | 结构性行情 |
| 题材轮动 | 默认 | 板块快速切换 |

### 2.4 仓位建议（自动映射）

| 大盘形态 | 建议仓位 | 操作策略 |
|---|---|---|
| 主升浪初/中期、底部反转、中级整理 | 70~90% | 正常做单（趋势票主菜） |
| 主升浪末期、加速期、顶部构筑、大箱体震荡 | 30~50% | 谨慎，只做最强主线 |
| 顶部确立、趋势下跌、加速杀跌、杀跌末期 | ≤20% | 观望，不做趋势票 |

### 2.5 量能判断

| 当日成交额 vs 20日均量 | 状态 | 影响 |
|---|---|---|
| > 1.2倍 | 活跃 | 趋势可信度↑ |
| 0.8~1.2倍 | 正常 | 中性 |
| < 0.8倍 | 缩量 | 警惕方向选择，少做 |

## 三、板块判断逻辑

### 3.1 4 窗口热度排名

每个概念板块计算：
- 今日涨幅
- 5 日累计
- 10 日累计
- 20 日累计

按各窗口分别排序，输出：
- 各窗口 TOP 5（强势）
- 各窗口 BOTTOM 5（弱势）

### 3.2 持续度评级（最关键）

> **同时进入多个窗口前列 = 真正的持续性主线**

| 持续度 | 标准 | 含义 |
|---|---|---|
| 4/4 ⭐⭐⭐⭐ | 4 个窗口全在 TOP 10 | 真主线，可放心做 |
| 3/4 ⭐⭐⭐ | 3 个窗口在 TOP 10 | 强主线 |
| 2/4 ⭐⭐ | 2 个窗口在 TOP 10 | 反弹或新主题 |
| 1/4 ⭐ | 1 个窗口在 TOP 10 | 一日游高风险 |

### 3.3 一日游识别

满足任一即认定为高风险一日游：
- 今日涨幅榜前 3，但 5 日累计跌幅 > 0
- 今日板块大涨，但板块内连板股 ≤ 2
- 板块龙头次日开盘高开 > 5% 但当日收阴
- 板块成交额相比 5 日均量放大 > 3 倍（透支）

### 3.4 退潮板块

满足以下任一：
- 5 日跌幅 > 3% 且 10 日跌幅 > 5%
- 连续 3 日缩量阴跌
- 主力资金连续 3 日净流出

### 3.5 板块共振用法

**个股阶段判断时**，若个股所在板块为：
- 🔥 强势主线（评级 ⭐⭐⭐ 以上）→ 评分 +8
- 🟢 持续走强 → 评分 +4
- 退潮 → 评分 -10
- 一日游 → 评分 -15

## 四、主力侦察五问

### Q1：是否有主力？

**满分 100，>= 60 分判定有主力**。

| 维度 | 权重 | 评分规则 |
|---|---|---|
| 换手率特征 | 30 分 | 1.5%~8% 适度 +30；过高/过低降权 |
| 股价稳定性 | 30 分 | 60 日日均振幅 < 4% +30；< 6% +20；> 6% +5 |
| 资金集中度 | 40 分 | 大单+超大单买入占比 > 35% +40；> 25% +25；< 25% +10 |

**输出**：
- 80+ 强主力
- 60~79 中等主力
- < 60 无明显主力

### Q2：主力性质判断

四选一：**机构主导 / 游资主导 / 机构+游资混战 / 散户为主**。

#### 机构特征（累加机构分）

| 维度 | 加分条件 |
|---|---|
| 振幅克制 | 30 日日均振幅 < 4% → +25 |
| 极端日少 | 30 日内涨跌 > 7% 的天数 ≤ 1 → +20 |
| 换手适度 | 30 日均换手 1.5%~4% → +25 |
| 慢牛节奏 | 创新高天数 > 8 + 暴涨天数 < 3 → +20 |

#### 游资特征（累加游资分）

| 维度 | 加分条件 |
|---|---|
| 振幅剧烈 | 30 日日均振幅 > 7% → +25 |
| 极端日多 | 30 日内涨跌 > 7% ≥ 5 天 → +30 |
| 换手偏高 | 30 日均换手 > 8% → +25 |
| 脉冲式 | 30 日内涨幅 > 5% ≥ 3 天 → +15 |

#### 综合判定

```
机构分 ≥ 60 且机构分 > 游资分 × 1.3  → 机构主导
游资分 ≥ 60 且游资分 > 机构分 × 1.3  → 游资主导
两者差距 < 20 且总和 > 80           → 机构+游资混战
其他                                  → 散户为主
```

### Q3：控盘程度（百分比）

三个维度加权：

```
控盘度 = 大单占比 × 0.5 + 控盘指标强度 × 0.3 + 价格集中度 × 0.2
```

| 控盘度 | 等级 | 含义 |
|---|---|---|
| ≥ 70% | 高度控盘 | 散户筹码很少，主力主导股价 |
| 50%~70% | 中度控盘 | 主力控盘但散户仍有筹码 |
| 30%~50% | 弱控盘 | 筹码分散 |
| < 30% | 无控盘 | 筹码混乱，主力难推动 |

#### 控盘指标公式

```
控盘指标 = (EMA(EMA(close, 13), 13) - 前一日值) / 前一日值 × 1000
```

> 双重 EMA(13) 平滑后的当日变化率。
> 留下来的只有"持续性资金推动"信号，过滤掉短期噪音。

| 控盘指标 | 含义 |
|---|---|
| > 0 且上升 | 主力推动中 |
| > 0 且下降 | 主力放慢 |
| < 0 且下降 | 主力撤退 |

## 五、个股阶段识别（11 细分）

### 阶段全景

```
   1. 吸筹期早    →  2. 吸筹期晚   →  3. 启动期
                                            ↓
   ←——————————    4. 拉升期初  →  5. 拉升期中  →  6. 拉升期加速
                          ↑              ↓                ↓
                          └———  7. 洗盘期   ←——————————┘
                                         ↓
                          8. 派发期早  →  9. 派发期中
                                              ↓
                          11. 底部反转  ←  10. 杀跌期
                                ↓
                                返回 1. 吸筹期早
   
                  ?  过渡期（信号混乱时归类）
```

### 各阶段判断规则

每个阶段都有 5~6 个特征，按累加分判断。**最高分阶段胜出**，分数 < 50 归过渡期。

#### 1. 吸筹期-早期（深度低位横盘）

| 特征 | 分值 |
|---|---|
| 距 60 日高点 < -20% | 25 |
| 30 日振幅 < 5% | 25 |
| 量能萎缩 | 20 |
| 控盘指标 < 0.5（刚要转正） | 15 |
| 60 日涨跌 \|x\| < 10% | 15 |

**主力动作**：悄悄收集筹码  
**操作**：观察等待，不入场

#### 2. 吸筹期-后期（建仓尾声）

| 特征 | 分值 |
|---|---|
| 距前高 -10% ~ -30% | 20 |
| 30 日振幅 < 6% | 20 |
| 控盘 0~1.5 上升 | 25 |
| 站稳 MA20 附近（距 \|x\| < 5%） | 15 |
| 量能温和 | 10 |
| 20 日资金累计正但 5 日震荡 | 10 |

**主力动作**：完成建仓，准备启动  
**操作**：观察 + 小试仓 (B 级)

#### 3. 启动期

| 特征 | 分值 |
|---|---|
| 突破近 30 日平台高点（涨幅 2%~8%） | 35 |
| 放量突破 | 25 |
| 控盘转正 + 站上 MA20 | 20 |
| 近期资金转正 | 20 |

**主力动作**：试盘、突破前期高点  
**操作**：A 级试仓

#### 4. 拉升期-初期（趋势刚确立）

| 特征 | 分值 |
|---|---|
| 多头排列 + 距 MA60 0~10% | 25 |
| 距 MA20 0~5%（刚突破） | 20 |
| 5 日和 10 日资金均正 | 20 |
| 接近 60 日新高 | 15 |
| 控盘转正 | 20 |

**主力动作**：确立趋势，吸引跟风  
**操作**：A 级建仓 30~50%

#### 5. 拉升期-中期（最舒服阶段）

| 特征 | 分值 |
|---|---|
| 多头排列 + 距 MA60 5%~30% | 25 |
| 资金 5/10/20 日均正 | 25 |
| 控盘 > 0 且上升 | 15 |
| 量价健康（量比 1.0~2.5） | 15 |
| 多头 + 近期创新高 | 20 |

**主力动作**：拉抬-洗盘-再拉抬  
**操作**：A 级持有 + 回踩加仓

#### 6. 拉升期-加速期

| 特征 | 分值 |
|---|---|
| 多头 + 创新高 + 距 MA60 > 20% | 25 |
| 量能放大 + 60 日涨幅 > 30% | 25 |
| 控盘 > 0 且上升 | 20 |
| 近 5 日涨幅 > 10% | 30 |

**主力动作**：拉升加速，吸引最后一批跟风盘  
**操作**：持有但警惕，目标位减仓

#### 7. 洗盘期（拉升中段健康调整）

| 特征 | 分值 |
|---|---|
| 60 日涨幅 > 15% + 多头 + 距 MA60 > 5% | 30 |
| 距前高 -5% ~ -15%（健康回撤） | 25 |
| 缩量回调 | 25 |
| 控盘仍为正 | 20 |

**主力动作**：震仓洗盘  
**操作**：持有，回踩可加仓（关键！）  
**警示**：跌破前期低点 + 放量 → 不是洗盘

#### 8. 派发期-早期（高位悄悄出货）

| 特征 | 分值 |
|---|---|
| 距 60 日高点 -5% ~ 0% + 多头 | 15 |
| 放量但近 5 日未创新高 | 25 |
| 10 日仍正但 5 日转负 | 25 |
| 控盘高位但开始下降 | 15 |
| 60 日涨幅 > 30%（前期有拉升） | 10 |

**前提**：必须有过明显拉升，否则不算  
**主力动作**：悄悄出货  
**操作**：减仓 30~50%

#### 9. 派发期-中期（加速派发）

| 特征 | 分值 |
|---|---|
| 距前高 -10% ~ -3% | 20 |
| 放量滞涨（疑似派发） | 30 |
| 20 日累计正但 5 日明显转负 | 25 |
| 控盘从高位下降 | 15 |
| 60 日涨幅 > 20% | 10 |

**主力动作**：加速出货  
**操作**：清仓离场

#### 10. 杀跌期

| 特征 | 分值 |
|---|---|
| 空头排列（价 < MA20 < MA60） | 25 |
| 资金三窗口均负 | 25 |
| 控盘 < 0 且下降 | 20 |
| 已跌破 MA60 | 20 |
| 阴跌缩量 | 10 |

**主力动作**：已撤退，让散户自相踩踏  
**操作**：清仓远离，不抄底

#### 11. 底部反转苗头

| 特征 | 分值 |
|---|---|
| 长期下跌后（60 日涨幅 < -10% + 距 120 日高 < -20%） | 20 |
| 控盘 < 0 但开始回升 | 25 |
| 近 5 日资金转正（10 日仍负） | 25 |
| 缩量后温和放量 | 15 |
| 近 3 日出现长下影 K 线 | 15 |

**主力动作**：新主力开始试探  
**操作**：C 级仅观察，不入场。等阶段切换到 1/2/3 期再考虑

#### 过渡期

所有阶段最高分 < 50 时归类。**信号混乱期**，不操作。

### 阶段惯性

为避免日间震荡跳变：
- 阶段需要持续 ≥ 3 个交易日才"正式确立"
- 切换到新阶段需新阶段评分 > 旧阶段评分 + 15

## 六、机会等级 4 级分类

| 等级 | 触发条件 | 历史胜率 | 仓位 |
|---|---|---|---|
| 🟢 **A 级·拉升加仓** | 拉升期持续>5天 + 回踩MA20 + 缩量 + 资金健康 | 65~75% | 加仓 30~50% |
| 🟢 **A 级·右侧建仓** | 拉升期刚确立(≤8天) + 回踩MA20 + 资金转正 | 55~65% | 建仓 30~50% |
| 🟠 **B 级·吸筹试仓** | 吸筹后期 + 控盘转正 + 站稳MA20 | 45~55% | 试仓 20~30% |
| 🔵 **C 级·左侧观察** | 杀跌期 + 底部反转预警 ≥ 3 项 | 35~45% | **不入场，仅观察** |

### 优先级

`A 级·拉升加仓 > A 级·右侧建仓 > A 级·启动期 > B 级·吸筹 > C 级·观察 > 无机会`

同一只票同一时间只显示**一个最高等级**。

### A 级·拉升加仓的精确条件

```
phase = 拉升期 AND days_in_phase >= 5
AND has_main_force = True
AND 距 MA20 绝对值 < 5%
AND close > MA20
AND 量能缩窄（近5日均量 < 前期20日均量 × 0.85）
AND 资金 5/10/20 日均为正
```

### A 级·右侧建仓的精确条件

```
phase = 拉升期 AND 1 <= days_in_phase <= 8
AND has_main_force = True
AND (距 MA20 < 5% OR sub_phase = 拉升期-初期)
AND close > MA20
AND 资金 5 日和 10 日均为正
```

### B 级·吸筹试仓的精确条件

```
sub_phase = 吸筹期-后期
AND has_main_force = True
AND close > MA20
AND (控盘指标 > 0 OR 5 日资金为正)
```

### C 级·左侧观察的精确条件

```
sub_phase = 底部反转苗头
AND 还在杀跌/反转过程中
AND 趋势未确认
```

## 七、综合评分公式

满分 100，基础分 50，按各维度加减。

```python
score = 50  # 基础分

# 1. 主力强度（最多+15）
score += 主力强度 × 0.15

# 2. 阶段加分
phase_pts = {
    拉升期: +25,
    启动期: +20,
    洗盘期: +15,
    吸筹期: +5,
    底部反转: 0,
    派发期: -15,
    杀跌期: -25,
    过渡期: 0,
}
score += phase_pts[main_phase]

# 3. 子阶段微调
sub_pts = {
    拉升期-中期: +5,
    拉升期-初期: +5,
    拉升期-加速期: -5,    # 末期降权
    洗盘期: +5,           # 洗盘期机会
    派发期-中期: -10,
}
score += sub_pts.get(sub_phase, 0)

# 4. 控盘度
if 控盘度 >= 70: score += 8
elif 控盘度 >= 50: score += 4

# 5. 主力性质
if 机构主导: score += 5
elif 游资主导: score += 2

# 6. 板块共振
if sector_strong: score += 8

# 7. 大盘环境
if not market_uptrend: score *= 0.85

# 8. 机会等级加成
if opportunity == A级: score += 5

# 收口
score = max(0, min(100, score))
```

| 60~74 | 趋势完好但等买点 |

### 减仓/清仓信号梳理

| 强度 | 信号 | 动作 |
|---|---|---|
| **强** | 跌破 MA60 | 清仓 |
| **强** | 控盘指标连续 5 日 < 0 且下降 | 清仓 |
| **强** | 周线 MACD 死叉 | 清仓 |
| **中** | 跌破 MA20 且 3 日不收复 | 减 50% |
| **中** | 出现明显放量滞涨（非主升浪期间） | 减 50% |
| **中** | 资金从持续流入转为持续流出 | 减 30% |
| **弱** | 量价健康度跌破 1.0（非主升浪） | 减 30% 或观察 |

### 主升浪期间的特殊处理

**主升浪进行中**（满足全部）时：
- 今日主力净流入 > 流通市值 × 0.5%
- 5 日累计 > 流通市值 × 1%
- 10 日累计 > 流通市值 × 1.5%
- 已离开 MA20 5%+ 且接近近期高点

→ **抑制**"放量滞涨"和"量价不健康"信号  
→ 评分 +25（远高于普通拉升期 +20）

避免把主升浪的强势误判为派发。
